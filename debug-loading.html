Debug Loading Issue

Open your browser console and check for these messages:

  • "AuthProvider useEffect running"
  • "Initial session check: {session, error}"
  • "Loading user profile for: [user-id]" or "No session, setting loading to false"
  • "User profile loaded: [profile]" or "Error fetching user profile: [error]"
  • "Setting loading to false"

Common Issues:

  1. Supabase connection error: Check if VITE_SUPABASE_URL and VITE_SUPABASE_ANON_KEY are set correctly
  2. Database error: Check if the user profile exists in the database
  3. Infinite loop: Check if loadUserProfile is being called repeatedly
  4. Network error: Check if there are CORS or network issues

Quick Fixes:

  1. Check browser console for error messages
  2. Verify environment variables in .env.local
  3. Check Supabase dashboard for any errors
  4. Try refreshing the page
  5. Clear browser cache and cookies
